# Tutorials

End-to-end Jupyter notebooks executed in CI. Empty in v0.0.1; will be populated in Phase 6.

Planned tutorials:

- `01_run_sweep.ipynb` — generate a sweep manifest from a parameter space.
- `02_aggregate_summaries.ipynb` — run `spatialtissuepy` and build a Zarr ensemble.
- `03_fit_landscape.ipynb` — embed and cluster to produce a `Landscape` bundle.
- `04_visualize.ipynb` — UMAP scatter, state-colored trajectories.
